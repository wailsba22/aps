## ğŸš€ Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/wailsba22/apis.git
cd apis
```

2. Start a local server:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js (with http-server)
npx http-server -p 8000
```

3. Open your browser:
```
http://localhost:8000
```
