# API Explorer

A simple, modern explorer for free public APIs.

## Structure
- `index.html`: Main page
- `style.css`: Base styles
- `scroll-button.css`: Scroll to top styles
- `script.js`: App logic (loading, filtering, modal)
- `apis-database.json`: API data
- `assets/images/logo.svg`: Brand logo
- `assets/icons/favicon.svg`: Scalable favicon
- `assets/icons/site.webmanifest`: PWA manifest

## Favicons
`index.html` links a modern SVG favicon and a web manifest. Optionally add PNG fallbacks if needed.

## Development
Open `index.html` in a browser or serve locally with any static server.
